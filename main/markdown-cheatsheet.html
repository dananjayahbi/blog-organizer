Markdown Cheat Sheet

This is a quick reference guide for Markdown syntax. Use this to format your blog posts.

Basic Syntax

Element Markdown Syntax Renders As
Heading 1 # Heading 1

Heading 1
Heading 2 ## Heading 2

Heading 2
Heading 3 ### Heading 3

Heading 3
Bold **bold text** bold text
Italic *italicized text* italicized text
Bold and Italic ***bold and italic*** bold and italic
Blockquote > blockquote
blockquote

Lists

Element Markdown Syntax
Ordered List 
1. First item
2. Second item
3. Third item
Unordered List 
- First item
- Second item
- Third item
Nested Lists 
- First item
  - Indented item
  - Indented item
- Second item
Task List 
- [x] Completed task
- [ ] Incomplete task

Links and Images

Element Markdown Syntax
Link [link text](https://www.example.com)
Image ![alt text](image.jpg)
Linked Image [![alt text](image.jpg)](https://www.example.com)

Code

Element Markdown Syntax
Inline Code `code`
Code Block 
```
code block
```
Syntax Highlighting 
```javascript
console.log('Hello world!');
```

Tables

Markdown Syntax 
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
 
| Left     | Center   | Right    |
|:---------|:--------:|----------:|
| Aligned  | Aligned  | Aligned  |

Horizontal Rule

Markdown Syntax
--- or *** or ___

Advanced Tips

Element Markdown Syntax
Footnote Here's a sentence with a footnote. [^1]

[^1]: This is the footnote.
Strikethrough ~~The world is flat.~~
Escape Character \* Without the backslash, this would be a bullet in an unordered list.
